Ravenous is a React App utilizing Yelp's API that allows users to search restaurants by location and genre ranking results by best match, highest rated, and most reviewed. I ultimately broke the project down into four parts of production, first, creating the static components, second, passing information to those components, third, setting the state of the Ravenous components and lastly, interacting with the Yelp API.

I broke the components down into App, SearchBar, SearchResults, PlayList, TrackList and Track, building out the CSS files for each, writing the HTML inside the render methods, importing and exporting the necessary components, and adding the methods for Track Search, renaming a PlayList, saving a PlayList to Spotify, removing a Track and adding a Track to a PlayList. After this, I passed the state of a search results parameter through a series of components to render an array of tracks.
