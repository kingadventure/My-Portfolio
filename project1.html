Ravenous is a React App utilizing Yelp's API that allows users to search restaurants by location and genre ranking results by best match, highest rated, and most reviewed. I ultimately broke the project down into four parts of production, first, creating the static components, second, passing information to those components, third, setting the state of the Ravenous components and lastly, interacting with the Yelp API.

I broke the components down into App, Business, and BusinessList, built out the CSS files for each, wrote the HTML inside the render methods. After this I set up the import and export functions to pass information from the parent component (App) to the child components (BusinessList and Business). Once this was done, I started work on how the app would interact with the Yelp API. First I set the state of the search bar elements, then simulated a search query. At this time I added the Yelp component that would be able to complete the functionality of the query. The Yelp.js component contained the API key I registered to the app, the function for administering a search, and the links from my variable names to Yelps uris. This query would take the filters at the top of the page into account.

This was one of my first React apps and the one that generated the most excitement for me in my early days of learning to code. I hope you get the chance to check it out!
